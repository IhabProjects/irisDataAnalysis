# Iris Flower Classification

My first machine learning project. Uses the classic Iris dataset to classify flower species based on petal and sepal measurements.

## What it does

Trains and compares 6 classification algorithms (LR, LDA, KNN, CART, NB, SVM) and picks the best one to make predictions.

## Result

SVM came out on top with ~98% accuracy on cross-validation and 96.7% on the final test set.

## Stack

- Python 3.13
- scikit-learn
- pandas
- matplotlib

## Run it

uv run main.py
